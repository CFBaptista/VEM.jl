## [Unreleased]

### Added

- Abstract interface for a vortex blob: `AbstractVortexBlob`. ([#3])
- Concrete type for a Gaussian vortex blob: `GaussianVortexBlob`. ([#3])

- Abstract interface for an induced field: `AbstractInducedField`
- Concrete types for a velocity field and a vorticity field: `VelocityField` and `VorticityField`

- Function to compute the field value induced by a vortex blob at a target: `induce`.
- Multi-threaded CPU implementation for the direct summation of fields induced by a collection of vortex blobs at a collection of targets: `direct_sum` and `direct_sum!`.

- Advection of a collection of vortex blobs: `advection!`. ([#5])
